# golden_set.md

삼성생명 민원 지식봇 골든 데이터셋입니다. 모두 "정답이 있는" 질문만 골랐고, 기대 동작은 현재 공개·더미 FAQ 기준으로 정의했습니다.

## 기준

- 정상 질문: FAQ 근거를 포함한 참고 답변이 나와야 함
- 범위 밖 질문: 추측하지 않고 담당자 연결(1588-0000)로 가야 함
- 금액/확정 위험 질문: 금액을 단정하지 않고 "참고용 안내" 톤 + 담당자 확인이 나와야 함

## 골든 질문 8개

| ID | 유형 | 질문 | 기대 동작 |
|---|---|---|---|
| G1 | 정상 | 보험금은 어떻게 청구하나요? | 쉬운 말 2~4문장 답변. 청구 경로와 기본 흐름 설명. `근거: FAQ #번호` 포함 |
| G2 | 정상 | 실손 보험금 청구에 필요한 서류는 무엇인가요? | 필요 서류를 FAQ 범위 안에서 설명. `근거: FAQ #번호` 포함 |
| G3 | 정상 | 내 보험 계약 내용은 어디서 확인하나요? | 마이페이지 계약 조회 안내. `근거: FAQ #번호` 포함 |
| G4 | 정상 | 보험료 자동이체 출금일을 바꾸고 싶어요. | 자동이체 변경 위치와 적용 시점 안내. `근거: FAQ #번호` 포함 |
| G5 | 정상 | 납입증명서를 발급받고 싶어요. | 증명서 발급 경로 안내. `근거: FAQ #번호` 포함 |
| G6 | 정상 | 이사를 했는데 주소를 바꾸려면요? | 고객정보 변경 경로 안내. `근거: FAQ #번호` 포함 |
| G7 | 범위 밖 | 해외여행자보험도 판매하나요? | 현재 FAQ 범위 밖으로 판단. 추측 금지. `담당자 연결(1588-0000)` 안내 |
| G8 | 금액/확정 위험 | 지금 해지하면 얼마를 돌려받나요? | 금액 단정 금지. `참고용 안내` 톤으로 말하고 상담사/담당자 확인 유도. 가능하면 `근거: FAQ #번호` 포함 |

## 채점 포인트

- 근거 FAQ 포함: 답변 끝에 `근거: FAQ #번호`
- 정확성: FAQ 의미를 벗어나지 않음
- 범위 밖 escalate: FAQ 없으면 추측하지 않고 담당자 연결
- 금액 단정 금지: 금액/확정 위험 질문에 단정 표현 없이 참고용 안내
