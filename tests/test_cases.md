## Reviewer 재검토

- G1~G6 일반 질문도 답변이 다소 보수적으로 작성됨
  - FAQ만으로 충분한 질문인데도 `담당자 연결(1588-0000)` 문구가 자주 붙음
- 범위 밖 G7은 처리 자체는 맞지만, 내부 예측 분류는 보험 도메인과 무관한 카드 카테고리로 흔들릴 수 있음
  - 최종 답변은 맞아도, 분류기 신뢰성이 낮으면 검색 비용과 품질이 같이 흔들릴 수 있음

## 가장 먼저 고칠 1가지

- 분류기를 도메인 우선 방식으로 고치기
  - 현재는 보험 질문이나 범위 밖 질문에서도 카드성 카테고리가 먼저 튀는 경우가 있음
  - 우선순위:
    1. 보험/카드 도메인을 먼저 가볍게 판별
    2. 그 다음 해당 도메인 안에서만 카테고리 분류
    3. 분류 점수가 애매하면 지금처럼 전체 FAQ fallback
  - 이유:
    - 이번 골든셋은 통과했지만, 가장 큰 잠재 리스크는 답변 생성보다 "잘못된 분류 출발점" 쪽에 있음
    - 이 부분을 먼저 잡으면 FAQ 검색 안정성과 근거 품질이 같이 좋아짐
